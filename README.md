# Aspio Tasks

Multi-workspace task management. Built for the Aspio fullstack take-home.

## Stack

| Layer          | Tech                                                |
| -------------- | --------------------------------------------------- |
| Framework      | Next.js 16 (App Router, Server Components)          |
| Language       | TypeScript (strict)                                 |
| Styling        | Tailwind CSS v4 with custom design tokens           |
| UI             | Hand-rolled primitives on Radix UI                  |
| Backend        | Supabase (Postgres, Auth, Realtime, Edge Functions) |
| Data fetching  | Server Components + SWR                             |
| Forms          | React Server Actions                                |
| Deployment     | Vercel                                              |

## Run locally

```bash
pnpm install
cp .env.example .env.local                              # paste Supabase URL + anon key
psql "$SUPABASE_DB_URL" -f schema.sql                   # or run schema.sql in the Supabase SQL editor
supabase link --project-ref <YOUR_REF> && pnpm db:types
pnpm dev
```

Open <http://localhost:3000>.

## Project layout

```
src/
  app/              App Router routes
  components/ui/    design-system primitives
  components/       feature components
  lib/
    supabase/       server, browser, and proxy clients
    utils.ts        cn helper
    env.ts          zod-validated env
  types/
    database.ts     regenerated by `pnpm db:types`
supabase/
  functions/        Edge Functions
schema.sql          tables + RLS + seed, runnable in one execution
```

## Scripts

| Script           | Purpose                            |
| ---------------- | ---------------------------------- |
| `pnpm dev`       | Next.js dev server                 |
| `pnpm build`     | Production build                   |
| `pnpm start`     | Serve a production build           |
| `pnpm lint`      | ESLint                             |
| `pnpm typecheck` | `tsc --noEmit`                     |
| `pnpm db:types`  | Regenerate `src/types/database.ts` |
